# Task 6: Sales Trend Analysis Using Aggregations

## Objective
Analyze monthly revenue and order volume using SQL.

## Tools Used
- SQLite (DB Browser for SQLite)

## Dataset
- orders table
- Columns: order_id, order_date, amount, product_id

## SQL Concepts Used
- GROUP BY year and month
- SUM(amount)
- COUNT(order_id)
- ORDER BY

## Outcome
Identified monthly sales trends and order volumes.
